# Installation

* Clone the repo

    ``` bash
    git clone https://github.com/Shakileash5/E-commerce-Django
    ```

* Install the Dependencies

    ``` bash
    pip install -r requirements.txt
    ```
* Start the server
    ``` bash
    python manage.py runserver
    ```
